getViews.js - An application that when configured with a channel ID and an (X) number of videos, returns the most views of the most recent (X) number of videos. 

To run getViews, you need node.js installed. This can be done by going to https://nodejs.org/en/download/

You must also download views.zip, and unzip it. 

Edit the file with any relevant details (api key, channel ID, amount of videos).

Go to the working directory in your command line, and then run the command 'node getViews.js'.

The command line should then be populated with the views of the videos, shown exactly like the example output given in the instructions.
